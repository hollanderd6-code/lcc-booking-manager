Réponses automatiques

Activez pour que le système réponde automatiquement aux questions fréquentes basées sur les informations ci-dessus

Lorsqu'activé, le système répondra automatiquement aux questions comme "À quelle heure je peux arriver ?", "Y a-t-il des draps ?", etc. Si la réponse n'est pas trouvée, vous recevrez une notification.

Questions-réponses personnalisées

Ajoutez vos propres paires question/réponse. Le bot répondra en priorité avec ces réponses si les mots-clés correspondent.

Ex : Mots-clés "piscine, nager, pool" → Réponse "Oui, le logement dispose d'une piscine privée !"

Ces messages apparaîtront comme raccourcis cliquables dans le chat.
