Depuis le menu Messages, retrouvez toutes les conversations avec vos voyageurs, quelle que soit la plateforme (Airbnb, Booking.com, BHGuest…). Cliquez sur une conversation pour lire et répondre. L'onglet Templates vous permet de créer vos messages automatiques.

💡 Les badges colorés indiquent la plateforme de chaque réservation : Airbnb, Booking.com ou BHGuest (réservation directe)
2
Créez un nouveau template
Titre, logements et moment d'envoi

Dans l'onglet Templates, cliquez sur « Nouveau template ». Donnez-lui un titre explicite, sélectionnez les logements concernés (un ou plusieurs), puis choisissez le déclencheur : à la réservation, avant l'arrivée, jour J à 7h, après l'arrivée, avant ou après le départ.

⏰ Le déclencheur définit quand le message sera envoyé. Choisissez par exemple « Avant l'arrivée » pour envoyer les instructions d'accès 24h avant le check-in.
3
Définissez la condition d'envoi
Filtrez selon la caution ou la plateforme

La condition d'envoi vous permet d'affiner encore plus : envoyer le message toujours, seulement si la caution est active, seulement si la réservation vient d'Airbnb, de Booking.com ou en direct. Très utile pour envoyer un rappel de caution uniquement aux voyageurs qui ne l'ont pas encore payée.

🔒 Pour envoyer automatiquement le lien de caution, sélectionnez « Seulement si caution non payée » — le message ne partira que si le voyageur n'a pas encore réglé sa caution.
4
Rédigez le message avec les raccourcis
Personnalisation automatique + traduction en 26 langues

Rédigez votre message en français et utilisez les raccourcis pour personnaliser automatiquement chaque envoi : prénom du voyageur, nom du logement, dates, code d'accès, WiFi, lien du livret d'accueil… Cliquez sur un raccourci pour l'insérer dans le texte.

🌍 Traduction automatique activée — vous rédigez en français, Boostinghost traduit automatiquement chaque message dans la langue du voyageur au moment de l'envoi (26 langues disponibles). Pas besoin de créer un template par langue.
